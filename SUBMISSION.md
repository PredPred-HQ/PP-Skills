## 项目简介 (300 字符)

PredP.red AI Skill 是集成 x402 支付协议的 AI 交易助手，让用户在 OpenClaw 中通过自然语言即可交易 predp.red 去中心化预测市场。使用 X Layer 实现低 Gas 费和快速确认，结合 OnchainOS API 提供完整的链上交互能力。

## 核心功能

### 1. 市场查看
- 热门市场列表展示
- 市场详细信息查询
- 实时赔率和流动性数据
- AI 市场分析和洞察

### 2. 买入操作
- 自然语言指令解析
- x402 支付授权签名
- 智能合约 buy() 调用
- 交易广播和实时追踪

### 3. 卖出操作
- 持仓查询和收益计算
- 部分/全部卖出选项
- x402 支付处理
- 交易确认和收益分配

### 4. AI 增强
- 意图识别和参数提取
- 风险评估和提示
- 智能市场推荐
- 友好的错误处理

### 2. AI 自然语言处理
- 基于大模型的意图识别

- 上下文感知对话
- 多轮交互支持

### 3. 智能合约交互
- ethers.js v6 集成
- 完整的 ABI 封装
- Gas 估算和优化
- 交易模拟和验证

### 4. X Layer 优势

## 安装和使用

### 前置条件
- Node.js >= 18.x
- OKX API 凭证
- X Layer 钱包
- OpenClaw 环境

### 安装步骤
```bash
# 克隆项目
git clone [你的仓库链接]
cd xlayer-hackathon

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API 凭证

# 编译项目
npm run build

# 运行演示
npm run dev "查看 predp.red 热门市场"
```

### 使用示例
```bash
# 查看市场

npm run dev "在市场 #123 买入 100 USDT 的是"

npm run dev "卖出市场 #123 的所有持仓"
```

系统提示词架构包含：
1. **角色定义**: 专业预测市场交易助手
2. **能力范围**: 市场查询、买入、卖出、持仓查看

- 执行智能合约调用
