# PredP.red AI Skill - X Layer Onchain OS Hackathon 项目规范

## 项目概述

**项目名称**: PredP.red AI Skill  
**项目定位**: 基于 x402 支付协议的 AI Skill，使 OpenClaw 用户能够直接调用 predp.red 去中心化预测市场智能合约

### 核心价值
- **无缝集成**: 用户无需离开 AI 对话界面即可参与预测市场交易
- **x402 支付**: 使用 X Layer 原生的 x402 支付协议处理交易费用
- **AI 辅助决策**: 结合 AI 分析提供市场洞察和交易建议
- **一键交易**: 简化 buy/sell 操作，降低用户参与门槛

## 技术架构

### 核心组件

1. **AI Skill Layer (OpenClaw Plugin)**
   - 自然语言理解与意图识别
   - 交易参数解析与验证
   - AI 驱动的市场分析与建议

2. **x402 Payment Layer**
   - 支付授权签名 (TEE)
   - 交易费用管理
   - 支付状态追踪

3. **Smart Contract Interface**
   - buy(positionId, amount, outcome) - 购买预测份额
   - sell(positionId, amount) - 卖出预测份额
   - getMarketInfo(marketId) - 获取市场信息
   - getUserPositions(user) - 获取用户持仓

4. **X Layer Integration**
   - 使用 OnchainOS API (okx-onchain-gateway)
   - 交易广播与追踪
   - Gas 估算与交易模拟

### 技术栈
- **AI Framework**: OpenClaw Skills
- **Blockchain**: X Layer (OKX Layer 2)
- **Payment Protocol**: x402
- **Smart Contract**: 已有的 predp.red 合约
- **API**: OKX OnchainOS Skills API

## 功能规格

### 1. 市场查看功能
**用户指令示例**:
- "查看 predp.red 的热门市场"
- "显示市场 #123 的详细信息"
- "我有哪些持仓？"

**实现细节**:
- 调用 `getMarketInfo()` 获取市场数据
- 展示赔率、流动性、截止时间等关键信息
- AI 分析市场趋势和热度

### 2. Buy 操作
**用户指令示例**:
- "在市场 #123 买入 100 USDT 的'是'份额"
- "我想投资这个预测，买 50 块"

**实现细节**:
- 解析市场 ID、金额、预测方向
- 调用 x402 完成支付授权
- 调用合约 `buy()` 接口
- 广播交易到 X Layer
- 返回交易确认和持仓信息

**流程**:
```
用户指令 → 参数解析 → x402 支付授权 → 合约调用 → 交易广播 → 确认返回
```

### 3. Sell 操作
**用户指令示例**:
- "卖出我在市场 #123 的所有份额"
- "平仓，卖出 50% 的持仓"

**实现细节**:
- 查询用户当前持仓
- 计算可卖出数量和预期收益
- 调用 x402 支付授权
- 调用合约 `sell()` 接口
- 广播交易并确认

### 4. AI 增强功能
- **市场洞察**: 基于历史数据和市场情绪提供 AI 分析
- **风险评估**: 提醒用户潜在风险
- **收益计算**: 实时计算潜在收益/损失
- **智能推荐**: 根据用户偏好推荐市场

## 安全考虑

### x402 支付安全
- 使用 TEE (可信执行环境) 签名
- 每次交易需用户确认
- 设置支付上限和频率限制

### 交易安全
- 交易前模拟 (simulation)
- Gas 估算和滑点保护
- 重入攻击防护

### 用户保护
- 清晰的风险提示
- 交易确认步骤
- 持仓和盈亏透明展示

## X Layer 集成点

### 使用的 OnchainOS Capabilities
1. **x402 Payments**: 支付授权和费用管理
2. **Wallet API**: 钱包连接和签名
3. **Trade API**: DEX 交易和流动性
4. **Market API**: 市场数据和行情

### 技术优势
- **低 Gas 费**: X Layer 的优化 EVM 提供极低交易成本
- **高性能**: 支持 5000 TPS，快速确认
- **安全性**: 基于 Optimism Stack 的欺诈证明机制
- **EVM 兼容**: 无缝部署现有以太坊合约
